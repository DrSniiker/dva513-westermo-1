
# pip install uv

# Activate the environment
* Linux:
```bash
source .\.venv\Scripts\activate
```

* Windows (powershell):
```ps1
.\.venv\Scripts\activate
```

* Windows (cmd):
```cmd
.\.venv\Scripts\activate
```

* Run a script within the environment
```bash
uv run main.py
```

# Deactivate
```bash
deactivate
```
